github.io Blog with Hugo!


SSG 중 Hugo를 사용하여 블로그를 만들었다. 처음에 뭐가 뭔지 몰라서 뚱땅뚱땅 만들었다. 이제 배포를 해야하는데 어떻게 하더라…

github.io 만들기

Install Hugo

brew install hugo hugo version

사이트 폴더 만들기

hugo new site blog

테마 붙이기

테그, 카테고리 잘보이는 테마가 안보여서 그나마 stack 테마가 제일 내가 원하는 템플릿이었다…

cd blog
git init
git submodule add https://github.com/CaiJimmy/hugo-theme-stack/ themes/hugo-theme-stack

config.toml파일 삭제

템플릿 데모 사이트 참고해서 수정하기

content 폴더 내용을 복사해와서 사용.

  • 포스트 생성 hugo new post/포스트제목.md 제목은 title 항목으로 고칠 수 있다.

확인

hugo server (-D) 웹서버 실행해서 수정사항 확인

host on github
